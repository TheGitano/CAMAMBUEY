FB CAMAM PRO - PWA (Brillante)

Archivos incluidos:
- index.html
- app.js
- manifest.webmanifest
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

Cómo usar localmente:
1. Sube los archivos a un hosting (puede ser GitHub Pages, Netlify, Vercel, o un servidor propio).
2. Abre la URL en el móvil (Android o iPhone).
3. En Android: desde el navegador (Chrome), el sistema ofrecerá 'Agregar a pantalla de inicio', o usar el menú -> 'Agregar a pantalla de inicio'.
4. En iPhone: abre en Safari y usa 'Compartir' -> 'Añadir a pantalla de inicio' (iOS tiene limitaciones en service workers, pero la PWA funcionará como una app web).

Notas:
- El app abre directamente búsquedas en Facebook (URL construida hacia facebook.com).
- Para funcionar como app completa (sin dominio), súbelo a un hosting con HTTPS.
- Si quieres, puedo desplegarlo por ti en GitHub Pages o Netlify y darte la URL lista para instalar.
